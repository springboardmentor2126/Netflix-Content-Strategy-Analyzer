# Netflix Content Analyzer
#Milestone 1 branch update
## 📌 Project Overview
This project analyzes the Netflix Kaggle dataset to understand content trends, genre distribution, country contributions, and rating patterns.

---

## 🎯 Project Scope
The goal of this project is to:
- Analyze Netflix content trends over time
- Identify most common genres and ratings
- Understand country-wise content distribution
- Prepare clean and normalized data for exploratory data analysis (EDA)

---

## ✅ Success Metrics
- Dataset cleaned (no duplicates, handled missing values)
- Categorical features normalized (genre, rating, country)
- Cleaned dataset saved for further analysis
- Ready for visualization and business insights

---

## 📊 Dataset
Source: Netflix Titles Dataset (Kaggle)
Total Records: 8807
Features: 12 columns
